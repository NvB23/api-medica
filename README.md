# Api Voll Med

### Sobre
Este projeto é uma api de agendamento de consultas médicas para clínicas hospitalares.

Foi implementado por [Naum Victor Batista](https://www.linkedin.com/in/naumvb/) 

### Ferramentas
The following guides illustrate how to use some features concretely:

* Spring Boot
* Spring Data JPA
* Spring Security
* Java JWT (Auth0)
* Spring Doc
* MySQL
* Flyway
* Lombok
